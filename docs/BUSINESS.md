## **PASO 1: ELIGE CÓMO COBRAR (Elige UNO)**

### **Opción A: Stripe (Recomendado - 5 minutos)**
```
✅ Ventajas:
- Fácil de configurar
- Cobra automáticamente cada mes
- Seguro y confiable
- Soporta tarjetas de crédito

❌ Desventajas:
- Cobra 2.9% + $0.30 por transacción

CÓMO EMPEZAR:
1. Ve a stripe.com
2. Haz clic en "Sign Up"
3. Completa datos básicos (nombre, email, país)
4. Verifica tu identidad
5. Conecta tu cuenta bancaria
6. ¡Listo! Puedes empezar a cobrar
```

### **Opción B: PayPal (Más fácil - 3 minutos)**
```
✅ Ventajas:
- Súper fácil
- Casi todos tienen cuenta PayPal
- Seguro

❌ Desventajas:
- Cobra 2.9% + $0.30 (igual que Stripe)
- Menos flexible

CÓMO EMPEZAR:
1. Ve a paypal.com
2. Crea cuenta (o usa la existente)
3. Ve a "Herramientas" → "Facturas"
4. Envía facturas a clientes
5. Ellos pagan online
6. Dinero llega a tu cuenta
```

### **Opción C: Transferencia Bancaria (Más seguro)**
```
✅ Ventajas:
- Sin comisiones
- Directo a tu banco

❌ Desventajas:
- Manual (tienes que recordar cobrar)
- Lento (2-5 días)

CÓMO EMPEZAR:
1. Dale tu número de cuenta al cliente
2. Envía factura por email
3. Espera a que transfieran
4. Verifica en tu banco
```

## **PASO 3: CÓMO COBRAR (El Flujo Real)**

### **Flujo Simple (Recomendado)**

```
CLIENTE DICE "SÍ"
    ↓
TÚ ENVÍAS FACTURA (por email)
    ↓
CLIENTE PAGA (Stripe/PayPal/Transferencia)
    ↓
DINERO LLEGA A TU CUENTA
    ↓

    ↓
CLIENTE USA EL ASISTENTE
```

### **Cómo Crear una Factura (5 minutos)**

**Opción 1: Usar Google Docs (Gratis)**
```
1. Ve a docs.google.com
2. Busca "Plantilla de Factura"
3. Elige una que te guste
4. Completa:
   - Tu nombre
   - Nombre del cliente
   - Fecha
   - Descripción (ej: "Suscripción Lobster Assistant - Mes de Mayo")
   - Precio
   - Tu número de cuenta (para transferencia)
   - O enlace de Stripe/PayPal
5. Descarga como PDF
6. Envía por email al cliente
```

**Opción 2: Usar Factura.com (Automático)**
```
1. Ve a factura.com (o equivalente en tu país)
2. Crea cuenta gratis
3. Completa datos de tu empresa
4. Crea factura en 2 minutos
5. Sistema envía automáticamente al cliente
6. Sistema registra el pago automáticamente
```

**Opción 3: Usar Stripe (Más profesional)**
```
1. En tu panel de Stripe
2. Ve a "Invoices"
3. Crea nueva factura
4. Agrega cliente y monto
5. Stripe envía automáticamente
6. Cliente paga directamente en Stripe
7. Dinero llega a tu cuenta
```
